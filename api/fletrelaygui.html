fletrelaygui.main module¶

This is the main file that can serve as a starting point for a Python console script.

Then run pip install . (or pip install -e . for editable mode) which will install the command run inside your current environment.

Besides console scripts, the header (i.e. until _logger…) of this file can also be used as template for Python modules.
